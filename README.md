# 🎯 Quiz / Bilgi Yarışması Oyunu

Bu proje, Python programlama dili kullanılarak geliştirilmiş çoktan seçmeli bir bilgi yarışması uygulamasıdır. Kullanıcıya sorular sorulur ve verilen cevaplara göre puan hesaplanır.

---

## 📌 Projenin Amacı

Bu projenin amacı, programlama dersinde öğrenilen temel kavramların (koşullar, döngüler, fonksiyonlar ve veri yapıları) uygulamalı olarak kullanılmasını sağlamaktır.

---

## 🚀 Özellikler

* 🧠 Çoktan seçmeli soru sistemi
* 📂 JSON dosyasından soru okuma
* 🎲 Rastgele soru seçimi
* 📊 Zorluk seviyesine göre soru dağılımı (kolay / orta / zor)
* ✅ Doğru ve yanlış cevap kontrolü
* 🏆 Skor hesaplama
* 💾 Skorları dosyaya kaydetme
* 🖥️ Tkinter ile grafiksel kullanıcı arayüzü

---

## 🛠️ Kullanılan Teknolojiler

* Python 3
* Tkinter (GUI)
* JSON (veri saklama)

## ▶️ Nasıl Çalıştırılır

1. Python yüklü olduğundan emin olun
2. Projeyi bilgisayarınıza indirin:

```
git clone https://github.com/esranuryaman/Bahar-D-nemi-Projesi-Quiz-Bilgi-Yar-mas-
```

3. Proje klasörüne girin:

```
cd Bahar-D-nemi-Projesi-Quiz-Bilgi-Yar-mas-
```

4. Programı çalıştırın:

```
python bilgi_yarismasi.py
```
